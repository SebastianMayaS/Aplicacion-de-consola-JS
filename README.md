## Descripción

Este proyecto consiste en una calculadora matemática con las 4 operaciones básicas, las cuales se ejecutan a través de un prompt. Los resultados de estas se podrán ver en la consola del navegador, así mismo el historial de todas las operaciones realizadas ordenadas de manera creciente (del 1 hacia adelante).


## Tecnologías Utilizadas

- Javascript
- HTML

## Uso del sitio

se tiene que abrir el archivo index.html, después se abre la consola del navegador que se este usando, los comandos para poder abrir la consola pueden variar segun el navegador, pero estos son los más usados:
- CTRL + SHIFT + J
- F12
- Click derecho y clickear en inspeccionar y se dirige a la pestaña Console
- Cmd + Option + C (MAC)
